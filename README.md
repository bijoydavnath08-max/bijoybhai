## Getting Started
1. **Open the site**
   - Simply double‑click `index.html` to view locally, or
   - Run a static server for a better experience (CORS, etc.).

2. **Run a dev server (optional)**
   ```bash
   # From the project root
   npm init -y   # creates a minimal package.json
   npm i -D http-server   # lightweight static server
   npx http-server . -p 8080
   ```
   Then open `http://localhost:8080` in your browser.

3. **Replace placeholder assets**
   - `placeholder-product.jpg` – add high‑quality product thumbnail images.
   - Brand logo files (`brand-*.png`) – add official brand logos.
   - Update the hero banner (`hero_banner_3_...png`) if you generate a new one.

4. **Customize content**
   - Edit `index.html` to change brand name, contact details, etc.
   - Extend `products` array in `js/main.js` with real catalog data or fetch from an API.
   - Adjust colors in `css/style.css` (primary: `#0d47a1` dark blue, accent: `#d32f2f` red).

5. **Deploy**
   - Upload the entire folder to any static‑hosting provider (Netlify, Vercel, GitHub Pages, etc.).
   - Ensure HTTPS and configure your domain.

## SEO & Performance Tips
- Add structured data (`<script type="application/ld+json">`) for product schema.
- Replace placeholder images with optimized WebP/AVIF formats.
- Enable gzip/ brotli compression on the server.

- Run Google PageSpeed Insights and address any suggestions.
